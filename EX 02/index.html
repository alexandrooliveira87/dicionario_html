Dicionário de Tags HTML

<h1> a <h6>

As tags de cabeçalho <h1> a <h6> são usadas para definir títulos e subtítulos. <h1> é o mais importante e <h6> é o menos importante.

<h1>Título Principal</h1>

<p>

A tag <p> é usada para definir um parágrafo de texto.

<p>Este é um parágrafo.</p>

<a>

A tag <a> é usada para criar um link. O atributo href especifica o destino do link.

<a href="https://www.exemplo.com">Visite o Exemplo</a>

<img>

A tag <img> é usada para incorporar uma imagem. O atributo src especifica o caminho da imagem.

<img src="caminho/para/imagem.jpg" alt="Descrição da Imagem">

<ul> e <li>

A tag <ul> define uma lista não ordenada, e <li> define um item da lista.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<ol> e <li>

A tag <ol> define uma lista ordenada, e <li> define um item da lista.

<ol>
  <li>Primeiro item</li>
  <li>Segundo item</li>
  <li>Terceiro item</li>
</ol>

<div>

A tag <div> é usada como um contêiner genérico para agrupar elementos e aplicar estilos CSS.

<div>Este é um contêiner</div>

<span>

A tag <span> é usada para estilizar uma parte do texto dentro de outro elemento.

<span style="color: red;">Texto em vermelho</span>

<table>, <tr>, <th> e <td>

A tag <table> é usada para criar uma tabela. <tr> define uma linha, <th> define uma célula de cabeçalho e <td> define uma célula de dados.

<table>
  <tr>
    <th>Cabeçalho 1</th>
    <th>Cabeçalho 2</th>
  </tr>
  <tr>
    <td>Dado 1</td>
    <td>Dado 2</td>
  </tr>
</table>
